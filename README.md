# vivire

Un diario personal, mínimo. Para anotar el día y recordar dentro de un año el mismo dia que hacias.

## Features

- Interfaz clara, tipografía cuidada
- Corrector ortografico en espanol
- Sonido opcional de teclado mecanico al teclear
- Entradas son privadas

## Nombre

*Vivire* viene de **vivir**.  
No es un planner ni un second brain.  
Es una forma de decir: esto paso hace un año hoy.
